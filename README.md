# 👀 Features

## 🔒 Auth

- Sign in with email and password
- Create a user with name, email, and password
- Log out of your account

## 🏠 Rooms

- Edit and delete rooms in which you are an admin
- View all rooms in which you are a participant
- If you are an admin, you can add a user to the room by email

## ✅ Bookings

- If you are an admin, you can book a room for certain time and date
- Other users in the room can sign up for that booking
- View the list of users who signed up for the bookings
